# Surgical Oncology Department KCCC Landing Page

This is a responsive single-page medical website built with React and Tailwind CSS, designed to match a specific visual reference.

## Features
- **Mobile-First Design**: Optimized for narrow desktop/mobile view (approx 520-600px width).
- **Custom Theme**: Uses specific colors (#f03b3b Red, #ffc928 Yellow) and fonts (Cairo, Inter).
- **Modular Components**: Broken down into Header, Hero, Workshops, Goals, etc.
- **Animations**: Subtle fade-ins, hovers, and pulses for a modern feel.

## Structure
- `index.html`: Main entry, Tailwind config, Fonts.
- `app.js`: Main layout.
- `components/`: Individual UI sections.
- `utils/`: Helper files (images).
